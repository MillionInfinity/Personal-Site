Boocalander

Is a friendly and easy to use book application.

Overview

Boocalander is a friendly and easy use single page app, designed to help users sort out which book they have checked, and show their due date. Users also can create, Add, edit and delete a book.

Tools

Vanilia js , bootstrap, grunt, npm , jquery, sass, Firebase
